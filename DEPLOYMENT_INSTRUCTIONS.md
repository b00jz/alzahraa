# How to Publish "Al Zahraa" to GitHub Pages
Since `git` is not installed on your computer, you can publish the site manually by uploading the files.

## Step 1: Create a GitHub Account
1. Go to [github.com](https://github.com) and Sign Up (if you don't have an account).

## Step 2: Create a New Repository
1. Log in to GitHub.
2. Click the **+** icon in the top right and select **New repository**.
3. Repository name: `al-zahraa` (or any name you like).
4. Select **Public**.
5. Check **Add a README file**.
6. Click **Create repository**.

## Step 3: Upload Your Files
1. In your new repository, click **Add file** > **Upload files**.
2. Drag and drop **ALL** files and folders from your `al zahraa` folder into the GitHub box.
   - Make sure to include `index.html`, `assets/` folder, etc.
3. Wait for them to upload.
4. Scroll down and click **Commit changes**.

## Step 4: Enable GitHub Pages (To make it a website)
1. In your repository, go to **Settings** (top tabs).
2. On the left sidebar, click **Pages**.
3. Under **Branch**, select `main` (or `master`) and folder `(root)`.
4. Click **Save**.
5. Wait a minute or two. GitHub will verify. refresh the page.
6. You will see a link like: `https://your-username.github.io/al-zahraa/`

**That is your live website link!** You can share it with your family.
